# 🚀 Equinox Nexus - Quick Start Guide

## Run in 2 Steps

### 1️⃣ Frontend Setup (Next.js)
```bash
npm install
npm run dev
```
✅ Open http://localhost:3000

### 2️⃣ Agent Setup (Optional but Recommended)
1. Get a GROQ API Key from [console.groq.com](https://console.groq.com)
2. Add it to your `.env.local` file:
   ```env
   GROQ_API_KEY=gsk_your_key_here
   ```

---

## What You'll See

- **Hero Dashboard**: Real-time migration opportunities
- **Interactive Globe**: Explore data by city
- **Agent Swarm**: Multi-agent debates on neighborhood selection
- **Simulation**: 5-year wealth trajectory analysis
- **Compliance**: Double-taxation treaty lookup

---

## Technical Architecture

- **Frontend**: Next.js with Framer Motion animations
- **AI**: Multi-agent orchestration using GROQ API
- **Data**: Real-time cost of living, AQI, and safety indices

---

**Total Setup Time**: ~2 minutes
